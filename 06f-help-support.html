Common Issues

Find solutions to frequently encountered problems and issues.

Why is my data not updating in real-time?

Real-time data updates depend on several factors:

  • Check your internet connection stability
  • Verify that real-time monitoring is enabled in Settings
  • Ensure your data sources are properly connected
  • Check if you've exceeded your API rate limits

If the issue persists, try refreshing the page or clearing your browser cache.

How do I reset my password?

To reset your password:

  1. Go to the login page
  2. Click "Forgot Password" link
  3. Enter your email address
  4. Check your email for reset instructions
  5. Follow the link and create a new password

If you don't receive the reset email, check your spam folder or contact support.

Why are my charts not displaying correctly?

Chart display issues can be caused by:

  • Browser compatibility: Ensure you're using a supported browser
  • JavaScript disabled: Enable JavaScript in your browser settings
  • Ad blockers: Temporarily disable ad blockers
  • Insufficient data: Some charts require minimum data points

Try refreshing the page or switching to a different browser if issues persist.

How do I export my data?

You can export data in several ways:

  • Dashboard exports: Use the export button on individual charts
  • Bulk export: Go to Data Tools â†’ Export to download large datasets
  • API access: Use our API endpoints for programmatic data access
  • Scheduled exports: Set up automated exports in the Data Tools section

Supported formats include CSV, Excel, JSON, and PDF for reports.

Can I customize my dashboard?

Yes! You can customize your dashboard in several ways:

  • Widget arrangement: Drag and drop widgets to rearrange
  • Chart types: Change visualization types for your data
  • Time ranges: Adjust default time periods for reports
  • Filters: Save custom filter combinations
  • Themes: Choose from available color themes

Changes are saved automatically to your user profile.
