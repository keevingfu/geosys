System Architecture

Technical Stack

Frontend

Vanilla JavaScript, HTML5, CSS3, ECharts 5.4.3 for visualizations, Font Awesome icons

Backend

Node.js, Express.js, JWT authentication, RESTful APIs, Express middleware

Database

PostgreSQL with specialized schemas for content, analytics, auth, and AI platforms

Containerization

Docker containers for API, database, and Nginx web server

Security

JWT tokens, role-based access control, rate limiting, CORS, Helmet.js

Deployment

Docker Compose for orchestration, GitHub auto-sync, CI/CD pipeline ready

Security Features

Authentication & Authorization

JWT-based authentication with role-based access control (admin, editor, user roles)

API Security

Rate limiting, CORS protection, Helmet.js headers, input validation, SQL injection prevention

Data Protection

Encrypted passwords, secure session management, audit logging, data sanitization

Infrastructure Security

Docker container isolation, environment variable management, secure credential storage
